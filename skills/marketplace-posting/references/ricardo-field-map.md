# Ricardo Field Map

A field map is reusable knowledge about Ricardo's listing form for a category or product type. It is not a scraper and not a bypass. It is a memory aid that prevents repeating slow exploratory clicks for similar one-by-one items.

## What To Capture

For every recurring category, record:

- item family, for example `porcelain service`, `bed frame`, `TV furniture`
- Ricardo category path selected
- sale format that worked
- required attributes revealed after category/product-type selection
- allowed values that matched our items
- fields that behaved unexpectedly
- photo-upload quirks
- validation messages and fixes
- whether the category is worth reusing for similar items

## How To Use It

1. Before opening Ricardo, match the item to the closest known field map.
2. Pre-fill the posting packet with known category and attribute values.
3. During form entry, use the map as a checklist.
4. If Ricardo changed the form or a better category is found, update the learning log after the session.
5. Promote repeated lessons from the learning log into this file.

## Known Maps

### Porcelain / Coffee Service

- Example item: `A0001_suisse-langenthal_kobalt_gold`
- Lesson: lead title with brand and product type, then piece count, color/decor, and `Vintage` when accurate.
- Color control behaved as single-select despite checkbox semantics.
- Main thumbnail matters strongly for search visibility; prepare it before publication.

### Bed Frame / Bedroom Furniture

- Example items: `A0005_holzbett_mit_federrost`, `A0006_biedermeier_kirschbaum_bett`
- Likely category: Home & Garden / Furniture / Bedroom / Beds, or the closest current Ricardo equivalent.
- Typical values: used condition, quantity 1, pickup only for bulky furniture.
- Required packet details: dimensions, included/excluded mattress/textiles, transport responsibility, pickup region.

### TV Furniture / HiFi Rack

- Example item: `A0007_glas_metall_tv_regal`
- Known packet category: `Moebel & Wohnen > Wohnzimmer > TV-Moebel`.
- Typical values: used condition, auction, pickup only due to glass/bulk.
- Required packet details: width, depth, height, material, condition, transport responsibility.

## Maintenance Rule

Do not guess hidden Ricardo internals. Record what was observed during normal verified-profile posting and keep uncertainty visible.
