# Shipping Rules

Default options:

- Pickup only: `Nur Abholung. Kein Versand.`
- Shippable: `B-Post Paket: CHF 15.00`, including handling effort.

Set pickup only if any are true:

- estimated weight over 10 kg
- any dimension over 50 cm
- fragile
- awkward shape
- annoying to pack safely
- likely to create buyer disputes if shipped
- furniture, rugs, large lamps, framed glass, heavy tools, large dinnerware sets

Set shippable only when all are true:

- under 10 kg
- under 50 cm in every dimension
- not fragile
- easy to pack
- value justifies the effort

If uncertain, recommend pickup only or ask the requesting user.
