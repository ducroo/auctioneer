# Packaging Rules

Choose how the item should be sold.

## Single Item

Default when:

- estimated market value is over CHF 50
- item is bulky
- item is distinctive, branded, antique, collectible, or easy to search
- furniture, rugs, vehicles, tools, electronics, art, and valuable decor

Examples:

- vintage chair
- branded drill
- rug
- complete high-quality dinnerware set

## Itemized Lot

Use for related mid-value items that are better sold together but should be described clearly.

Good when:

- items share a purpose, brand, style, or room
- individual selling would be tedious
- buyers still need counts/details

Examples:

- dinnerware set for 12
- mechanic wrench set, 15 pieces
- box of matching Christmas ornaments

## Concrete Box Lot

Use for low-value, high-volume items with a shared theme.

Swiss wording: use concrete descriptions, not mystery/surprise wording.

Examples:

- `Kiste diverses Werkzeug`
- `Kiste Küchenzubehör`
- `Kleiderpaket Damen Grösse M`
- `Box mit Dekoartikeln`

Avoid:

- `Surprise Box`
- `Mystery Lot`
- vague listings with no visible sample contents
