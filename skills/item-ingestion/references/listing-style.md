# Listing Style

Language: Swiss-style German.

Tone: concise, factual, buyer-friendly. Use bullets. Avoid hype.

## Required Opening Disclaimer

Put this first so nobody misses it:

```text
Abholung. Versand ist möglich, aber wir verrechnen CHF 15 zusätzlich nur fürs Handling, da es viel Aufwand ist.

Privatverkauf: Gekauft wie gesehen. Jede Gewährleistung, Garantie oder Rücknahme wird wegbedungen.
```

For pickup-only items, use this instead of the first line:

```text
Nur Abholung. Kein Versand.

Privatverkauf: Gekauft wie gesehen. Jede Gewährleistung, Garantie oder Rücknahme wird wegbedungen.
```

## Listing Shape

```text
Titel:
[Kurzer, suchbarer Titel]

Beschreibung:
- [Was ist es?]
- [Marke/Modell/Material, falls bekannt]
- [Zustand]
- [Masse / Grösse / Anzahl, falls relevant]
- [Lieferumfang]
- [Abholung oder Versand]
- [Preis / Mindestpreis, falls durch sales-strategy vorhanden]
```

## Placeholders

If mandatory information is missing, still draft the listing and insert placeholders exactly where needed.

Examples:

```text
- Masse: [BITTE MASSE EINGEBEN]
- Grösse: [BITTE GRÖSSE EINGEBEN]
- Zustand: [BITTE ZUSTAND BESTÄTIGEN]
- Anzahl: [BITTE ANZAHL EINGEBEN]
```

## Address And Contact

- Mention only the general pickup region if needed: `Abholung in der Region Rüfenacht/BE.`
- Do not include the full address before buyer commitment.
- Do not include phone number or personal email.
