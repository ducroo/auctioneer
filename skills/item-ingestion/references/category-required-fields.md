# Category Required Fields

Before drafting, decide the item category and check whether buyer-critical information is present.

Use these states:

- `known`: explicitly provided by Lucas or clearly readable.
- `visible_estimate`: estimated from photo; label as approximate.
- `missing_needs_lucas`: important and not visible.
- `not_applicable`: not needed for this item.

## Universal

- Item name / plain description.
- Condition: pristine, very good, good, shows wear, damaged, untested.
- Brand / maker / model if visible or relevant.
- Contents included.
- Obvious damage, missing parts, stains, chips, cracks, repairs.
- Shipping risk: bulky, heavy, fragile, awkward, annoying to pack.

## Clothes

- Size from label: S/M/L or EU size.
- Gender / target fit: Damen, Herren, Kinder, Unisex.
- Brand if visible.
- Condition and visible wear.
- Piece count if bundled.

## Shoes

- EU size.
- Gender / target fit.
- Brand if visible.
- Condition.
- Sole wear.

## Dinnerware

- Exact count by type: plates, bowls, cups, saucers, serving pieces, etc.
- Condition: chips, cracks, crazing, stains, wear.
- Maker / series if visible.
- Whether it is a complete set or mixed lot.

## Rugs

- Length x width in cm.
- Condition: stains, fading, edge wear, smell, holes.
- Material if known.
- Pickup/shipping feasibility.

## Furniture

- Length x width x height in cm.
- Condition: scratches, wobble, stains, structural damage.
- Material if known.
- Disassembly / transport notes.
- Pickup only unless clearly small and easy to pack.

## Electronics And Tools

- Tested or untested.
- Working condition.
- Brand and model number if visible.
- Included accessories.
- Battery/charger presence where relevant.
- Plug type / voltage if relevant.

## Art And Decor

- Dimensions in cm.
- Material / medium.
- Condition.
- Signature, stamp, or markings if visible.

## Concrete Box Lots

Use concrete wording such as `Kiste diverses Werkzeug`, not mystery/surprise wording.

- Common theme.
- Approximate contents.
- Number of pieces if practical.
- Condition range.
- What is definitely included.

## Vehicles

Vehicles need a separate structured checklist. Do not draft a vehicle listing from photos alone.
